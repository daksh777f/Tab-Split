# Tab Split - AI Powered Verdicts

A live hackathon project demonstrating autonomous knowledge graphs with Cognee Cloud.

## Structure
- `apps/web`: Next.js 15 frontend.
- `services/memory-engine`: FastAPI + Cognee backend.

## Getting Started
1. Run `npm install` in the root (installs concurrently).
2. Set up `.env` files in `apps/web` and `services/memory-engine`.
3. Install Python dependencies in `services/memory-engine`.
4. Run `npm run dev` in the root to boot both services.
